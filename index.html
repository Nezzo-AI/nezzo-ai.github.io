💡 Was ist Nezzo AI?

Nezzo AI ist eine Open-Source KI-Chat-Plattform, die du selbst hosten, anpassen und trainieren kannst. Starte kostenlos mit Free-Tier APIs oder verbinde dein eigenes Llama-Modell.

  • ✅ Chat-Interface mit Login (Email/Google)
  • ✅ Community: Chats teilen, liken, diskutieren
  • ✅ Tiers: Free • Ultra • GOD • Nezzo
  • ✅ Supabase für Auth, DB & Storage (Free Tier)
  • ✅ Bereit für Llama 70B, Qwen, Mistral & mehr

🏗️ Wie ist es aufgebaut?

Frontend

  • HTML/CSS/JS (eine Datei)
  • Responsive & Dark Mode
  • GitHub Pages Hosting

Backend

  • Supabase (Auth, PostgreSQL, Realtime)
  • Region: EU-West (DSGVO-freundlich)
  • Free Tier: 500MB DB, 1GB Storage

KI-Inferenz

  • Phase 1: Free APIs (Groq, HF, Together)
  • Phase 2: Eigenes Fine-Tuning (QLoRA)
  • Phase 3: Llama 70B via API oder lokal

🪙 Tokens & Rechen-Kosten

1.000 Tokens ≈ 750 Wörter
Input: ~0.15$/1M Tokens (Groq Free Tier)
Output: ~0.60$/1M Tokens (Groq Free Tier)

Beispiel: Ein Chat mit 2.000 Tokens kostet dich bei Free APIs ≈ 0€ (within limits). Bei eigenem 70B-Modell: ~0.02€ auf Spot-GPUs.

⚠️ Wichtig: Ein 70B Llama-Modell selbst zu trainieren kostet ~300-2000€ (GPU-Cluster).
✅ Aber: Du kannst mit 7B/8B-Modellen starten (0€ auf Colab Free) und später upgraden.

💡 Kosten sparen:

  • Nutze Free-Tier APIs für den Start (Groq, Together, HF)
  • Fine-Tune nur kleine Modelle (7B) mit QLoRA
  • Exportiere als GGUF → läuft sogar auf CPU
  • Hoste die Website auf GitHub Pages (0€)
